**Tic-Tac-Toe игра**

Минималистичная веб-игра "Крестики-нолики" с выдачей промокода победителю и интеграцией с Telegram-ботом

Проект выполнен в рамках тестового задания и демонстрирует:
— работу с фронтендом;
— взаимодействие с Telegram Bot API;
— простую связку web + Telegram без сложной архитектуры.

**Стек технологий:**
— Frontend: HTML5, CSS3, JavaScript
— Backend: Node.js, Express, Telegram Bot API

**Frontend:**
— Размещен через GitHub Pages, открывается по публичной ссылке и не требует сборки

**Backend:**
— Развернут как web-service через Render, работает постоянно и без локального запуска, использует long polling

Реализация осознанно упрощена, т.к. **скорость > качество**. Проект выполнен на понятную пользовательскую логику, простоту запуска для проверяющего, и минимальное количество зависимостей.
